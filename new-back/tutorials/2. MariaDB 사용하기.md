## MariaDB 사용하기

1. 데이터 베이스 접속 후 데이터베이스 확인하기
   - **show databases**;
2. 데이터베이스 생성
   - **create databases** example_db;

3. 데이터베이스 사용하기
   - **use** example_db;
   - **create table** user
   - **show tables**;
   - **alter table**, **insert into**, **select * from**  등 SQL문은 따로 학습이 필요함


4. 데이터베이스 삭제하기
   - drop database example_db;
